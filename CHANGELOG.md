# Changelog

## 0.1.0 - 2026-06-04

- Added a native collapsible **Linked Files** view section for Markdown files linked from the current Markdown file.
- Added Markdown Browser current-file tracking across browser navigation, custom previews, editor focus, and open Markdown tabs.
- Preserved linked-file fragments when opening shortcuts from the linked-files view.
- Added a Markdown Browser debug-info command for troubleshooting current-file and tab detection.
- Kept the existing Markdown files view focused on workspace file browsing, with tree/list modes and gitignored-file visibility controls.
